# ADIDAS 1-PAGER — Interactive Concept

This project was created as part of an internship application for adidas in response to the following prompt:

> **SHOW US WHO YOU ARE AND WHAT “YOU GOT THIS” MEANS TO YOU.**

---

## Concept

Rather than submitting a static PDF, I interpreted the 1-pager as an interactive digital experience.

This project transforms the idea of a single page into a scroll-driven storytelling journey, revealing different chapters of identity, growth, discipline, and mindset.

---

## Live Version

You can view the interactive version here:

https://sofijasof.github.io/onepager_SR/

---

## Technical Approach

Built using:
- HTML
- CSS 
- JavaScript

The layout uses a 3D camera model where each frame is positioned in depth space and revealed progressively through scroll.
